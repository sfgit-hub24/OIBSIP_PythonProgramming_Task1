# OIBSIP Python Programming Internship
## Task 1: BMI Calculator

### Description
A GUI-based Body Mass Index (BMI) Calculator developed using Python and Tkinter.  
The application calculates BMI based on user input, stores data in a CSV file, and visualizes BMI trends using graphs.

### Technologies Used
- Python
- Tkinter
- CSV
- Matplotlib
- Datetime

### Features
- User-friendly graphical interface
- BMI calculation using height and weight
- BMI category classification
- Stores BMI records in a CSV file
- View BMI history
- Graph visualization of BMI trends
